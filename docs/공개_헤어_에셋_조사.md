# 시중 공개 헤어 에셋 — SaaS 적합성 조사

> 질문: 다운로드해서 우리 라이브러리에 바로 쓸 수 있는 헤어 프리셋이 있는가?
> 작성일: 2026-05-05

---

## 결론 한 줄

> **"바로 다운로드해서 SaaS에 사용 가능한" 라이선스 깨끗한 anime VRM 헤어는 매우 드물다.** Sketchfab CC0/OpenGameArt 기반 5~10종 큐레이션 + VRoid Hub "redistribution=allow" 필터 5~10종은 가능하지만, 35종 라이브러리를 외부 에셋만으로 채우는 건 비현실적. **현실적 권장: 자체 제작 25종 + 외부 큐레이션 10종 하이브리드**.

---

## 1. SaaS의 라이선스 요구 조건 — 왜 까다로운가

우리 서비스는:
- 사용자가 결제하고 아바타 생성
- 헤어 메시가 사용자 VRM에 포함되어 다운로드
- 사용자가 그 VRM으로 VTuber 활동·수익화

→ 필요한 라이선스 권한:
1. ✅ **Commercial use (corporate + personal)** 모두 허용
2. ✅ **Redistribution (재배포)** 명시적 허용 — *제일 중요. 우리 SaaS가 사용자에게 메시를 전달하므로*
3. ✅ **Modification (수정)** 허용 — 우리 마스터에 맞게 변형
4. ⚠️ **Credit (크레딧)** 표기 의무 — 가능하지만 UI에 추가 필요

대부분 무료 헤어 에셋이 (1)만 허용하고 (2) **재배포는 금지**라서 SaaS에 부적합합니다.

## 3. 추천 큐레이션 소스 (실제 활용 가능)

### A. VRoid Hub — "redistribution=allow" 필터 큐레이션 (5~10종)

**검색 URL** (직접 접속해서 헤어만 골라보세요):
```
https://hub.vroid.com/en/license?
  allowed_to_use_user=everyone
  &corporate_commercial_use=allow
  &personal_commercial_use=allow
  &modification=allow
  &redistribution=allow
  &credit=unnecessary
```

이 조건을 모두 만족하는 모델은 전체의 **5% 미만**이지만, 거기서 헤어 부분만 추출해서 쓸 수 있습니다.

### C. BOOTH Extended Commercial License (유료) — 가장 빠른 길

[Extended Commercial License (mightbemagic)](https://booth.pm/en/items/2980775) 같은 패키지가 있고, 작가별 추가 비용으로 SaaS 재배포 권한을 얻을 수 있습니다.

**예상 비용:**
- 헤어팩 1종 단가 5,000~50,000원
- Extended Commercial 추가 라이선스 비용 1만~10만원
- 30종 채우려면 총 ~100~300만원

**시간 절감:** 35종 모두 자체 제작 (~80시간) 대비 약 30시간 절감 가능 (큐레이션·메일 협의 시간 포함).

**검색:**
- [Sketchfab CC0 헤어 검색](https://sketchfab.com/search?q=hair&licenses=322a749bcfa841b29dff1e8a1bb74b0b) (CC0 필터)
- [Sketchfab CC0 + 3D 모델](https://sketchfab.com/tags/cc0)

**확인된 후보:**

## 4. 라이선스 위험도 정리 — 5분 만에 판단

| 라이선스 표기 | SaaS 사용 가능? |
|---|---|
| **CC0 / Public Domain** | ✅ 무조건 OK |
| **MIT / Apache / GPL** (코드 라이선스) | ✅ OK (3D 자산에는 거의 적용 안 됨) |
| **CC BY** | ✅ OK (크레딧만 표기) |
| **CC BY-SA** | ⚠️ 주의 — Share-alike 조항으로 SaaS 전체가 같은 라이선스가 되어야 할 수 있음 |
| **CC BY-NC** | ❌ Non-commercial — 우리는 결제받음 → 부적합 |
| **CC BY-ND** | ❌ No-derivative — 우리는 마스터에 맞게 수정해야 함 → 부적합 |
| **VRM "Avatar use only"** | ❌ 본인 아바타 한정 → SaaS 재배포 부적합 |
| **"Personal use only"** | ❌ 명백히 부적합 |
| **"Commercial use OK + Redistribution OK + Modification OK"** | ✅ 우리에게 완벽 |

**작가 약관에 명시 안 된 사항이 있으면 → 작가에게 직접 메시지로 확인 + 답변 캡처 보관** (분쟁 시 증거).

## 5. 우리 프로젝트 권장 — 4가지 전략

### 전략 1: 100% 자체 제작 (시나리오 D, 이전 가이드)
- 시간: 80시간
- 비용: 0원
- 안전도: ★★★★★
- 추천도: ★★★★★ — 가장 안전

### 전략 2: 자체 제작 25종 + Sketchfab CC0 5종 + VRoid Hub 큐레이션 5종
- 시간: 60시간 + 큐레이션 5시간 + 작가 협의 5시간 = ~70시간
- 비용: 0원
- 안전도: ★★★★ — 큐레이션 항목 작가 확인 필요
- 추천도: ★★★★

### 전략 3: 자체 제작 20종 + BOOTH 유료 Extended 15종
- 시간: 40시간 + 큐레이션·구매·테스트 10시간 = 50시간
- 비용: 100~300만원
- 안전도: ★★★★★
- 추천도: ★★★ — 학생 프로젝트는 비용 부담

### 전략 4: AI 도구 + 자체 제작 하이브리드
- FiberShop(학생 무료) 또는 PixelCut 활용 → 빠르게 텍스처 생성
- Blender에서 카드 메시 생성 + 텍스처 적용
- 시간: 50시간
- 비용: 0~50만원
- 안전도: ★★★★ — 도구 약관 확인
- 추천도: ★★★★

## 7. 작가 메시지 템플릿 (참고)

### 영어 (Sketchfab/VRoid Hub 작가용)

> Hi! I'm building a SaaS web service called Virtual Avatar that lets users create their own 3D VTuber avatars from a photo. We'd love to use your hair preset as part of our preset library, where users can select your hair to attach to their avatar and download as a VRM file. This means the hair mesh would be **redistributed** to our paying users.
>
> Could you let me know:
> 1. Is this kind of SaaS redistribution use allowed under your license?
> 2. Do you require credit attribution? If so, where (in-app, on website, in metadata)?
> 3. Is there any additional license fee for this commercial SaaS use case?
>
> Thank you for your work!

### 일본어 (BOOTH 작가용)

> 初めまして、バーチャルアバター生成のSaaSウェブサービス「Virtual Avatar」を開発中の[이름]と申します。
>
> 弊社サービスでは、ユーザーが写真から3D VTuberアバターを生成し、ヘアスタイルをライブラリから選択してVRMファイルとしてダウンロードできるようにする予定です。つまり、貴方のヘアメッシュデータが**有料ユーザーに再配布される**形になります。
>
> 以下、ご確認いただけますでしょうか:
> 1. 貴方の現在のライセンスで、このようなSaaS再配布利用は可能でしょうか?
> 2. クレジット表記は必要でしょうか?(必要な場合、アプリ内/ウェブサイト/メタデータのいずれに記載すれば良いか)
> 3. 商用SaaS利用に対して追加ライセンス料は必要でしょうか?
>
> ご検討よろしくお願い致します。

## 8. 핵심 요약

> **시중 공개 헤어 에셋만으로 35종 라이브러리를 채우는 건 비현실적.** SaaS 재배포까지 명시 허용된 anime VRM 헤어는 매우 드물고, 작가별 약관 확인에 시간이 더 들 수 있음.

> **현실적 답: 자체 제작 25종 (이현지님 VRoid) + Sketchfab CC0/VRoid Hub 큐레이션 5~10종 = 30~35종.** 자체 제작이 라이선스·품질·통제 모두 우월하지만, 큐레이션 5~10종으로 다양성 보강 가능.

> **단, 큐레이션은 "작가에게 SaaS 사용 명시 + 서면 확인 + 캡처 보관"이 필수.** 한국·일본 모두 라이선스 분쟁 시 작가가 우위.
